# 🎓 VIP Study Bot v6.0

Telegram bot for extracting videos & PDFs from CareerWill, SelectionWay & Study IQ.

## Deployed on Render

## Commands
- `/start` - Start the bot
- `/help` - Show help
- `/cancel` - Cancel operation

## Features
- 🎯 CareerWill Extract
- 🏆 SelectionWay Extract  
- 📘 Study IQ Extract (With/Without Login)
